# Replay Assistant Online Beta Checklist

## Before each test

- Open the Render website a minute early if using the Free instance.
- Create a new private session.
- Confirm every camera shows the same session code.
- Confirm permission to record every athlete.

## Controller tests

- Standard mode with the controller camera
- Coach Controller Mode on iPad without using its camera
- Record, Replay, and Save Replay
- White 3-2-1-GO countdown
- Red recording countdown
- Green Replay Ready message
- Playback at 0.25x, 0.50x, 0.75x, and 1x
- Drawing tools, undo, clear, and eraser
- Download Original 1x
- Save thumbnail and Choose Frame

## Camera tests

- Scan QR code on iPhone Safari
- Open camera link on Android Chrome
- Stop and restart a camera
- Lock and unlock the phone
- Turn Wi-Fi off and back on
- Confirm the same camera number returns after reconnecting

## Session privacy test

- Create two private sessions in separate browsers.
- Add one camera to each session.
- Confirm cameras never appear in the other session.

## Network tests

- Same Wi-Fi
- Different Wi-Fi networks
- Camera on cellular data
- School or gym guest Wi-Fi

If same-network tests work but cross-network tests fail, configure TURN before wider public testing.
